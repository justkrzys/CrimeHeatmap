# Chicago PD Crime Heatmaps
Demonstration of semi-Big Data heatmap graphing using data from Chicago PD databases.

## Usage
The program takes three different Chicago PD crime database files as input, hosted [here](https://drive.google.com/drive/folders/1ibIPqYtDuX18PN-Tlj_imSMa4gpSrzRi?usp=sharing). These files contain the data used to generate the heatmaps. Generation of the heatmaps may be configured to the user's liking.
